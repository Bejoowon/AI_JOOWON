# 바이브 코딩 실전 가이드

> **이제 말로 시키는 시대입니다.**

---

## 왜 바이브 코딩인가

2022년 ChatGPT가 나오고 AI가 생활 속으로 들어왔지만, 우리는 여전히 "이거 뭐야?", "이거 설명해줘" 수준으로만 AI를 사용했습니다.

그런데 세상이 바뀌었습니다.

단순 질문을 넘어, **내가 의도한 대로 실행하고 일을 처리하는 AI** — 에이전트의 시대가 도달했습니다. 사용자는 AI가 해온 업무를 보고, 다시 지시 내리고, 결과를 받아봅니다.

에이전트에게 일을 시켜본 사람은 단언컨대, 이전 방식으로 돌아가기 어렵습니다.

::: tip 바이브 코딩은 마법봉 같습니다
프롬프트 몇 자만 입력하면 떡하니 결과물을 내놓습니다. 하지만 마법의 결과물이 항상 완벽한 건 아닙니다.

이제는 AI의 성능이 좋아져서 AI를 내 컴퓨터로 초대하고, 내 파일을 읽게 하고, 내 일을 시키는 방식으로 진화했습니다. 확률에 의존하던 프롬프팅에서 **가이드를 내리고, 지시하고, 결과를 받아보는 방식**이 된 겁니다.
:::

이 가이드를 따라오고 나면, 마치 신입사원을 데려다가 일을 가르치고, 전문가를 데려와서 나의 프로젝트를 같이하는 느낌을 받게 될 겁니다.

## 이 가이드에서 다루는 것

이 가이드는 **클로드 코드(Claude Code)** 사용을 기준으로 합니다. 바이브 코딩의 개념은 다른 AI 도구에도 동일하게 적용됩니다.

| 파트 | 내용 |
|------|------|
| **1. 사전가이드** | 환경 설정 — GitHub, Claude Pro, VS Code, Claude Code 설치 |
| **2. 스킬 개념** | 스킬이란 무엇인지, 직접 만들고 배포하기 |
| **3. 바이브코딩 기초** | CLAUDE.md, MCP 등 핵심 개념과 실습 |

::: info 코딩 경험이 없어도 됩니다
"터미널"이 뭔지 몰라도, "AI"라는 단어가 아직 어색해도 괜찮습니다. 이 가이드는 완전 초보자를 위해 초기 환경 설정부터 다루고 있습니다.
:::
