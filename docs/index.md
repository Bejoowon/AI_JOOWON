# 바이브 코딩 실전 가이드

> **코드를 몰라도, 터미널을 몰라도 괜찮습니다. 중요한 건 무엇을 시킬지 아는 것입니다.**

AI에게 질문하던 시대에서, **AI에게 일을 시키는 시대**로 바뀌었습니다.

이 가이드를 따라가면 AI를 내 컴퓨터로 초대하여, 내 파일을 읽게 하고, 내 일을 시키는 방법을 배울 수 있습니다. 마치 신입사원에게 일을 가르치고, 전문가를 데려와 나의 프로젝트를 같이하는 느낌 — 그것이 바이브 코딩입니다.

---

## 이 가이드에서 다루는 것

이 가이드는 **클로드 코드(Claude Code)** 사용을 기준으로 합니다. 바이브 코딩의 개념은 다른 AI 도구에도 동일하게 적용됩니다.

| 파트 | 내용 |
|------|------|
| **1. 사전가이드** | 환경 설정 — GitHub, Claude Pro, VS Code, Claude Code 설치 |
| **2. 클로드코드란** | Claude Code가 뭔지, 기존 AI와 뭐가 다른지 |
| **3. 스킬 개념** | 스킬이란 무엇인지, 직접 만들고 배포하기 |
| **4. 바이브코딩 기초** | CLAUDE.md, MCP 등 핵심 개념과 실습 |

::: info 코딩 경험이 없어도 됩니다
"터미널"이 뭔지 몰라도, "AI"라는 단어가 아직 어색해도 괜찮습니다. 이 가이드는 완전 초보자를 위해 초기 환경 설정부터 다루고 있습니다.
:::
