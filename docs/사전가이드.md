# 사전준비 가이드

세미나 전날까지 이 페이지의 내용을 완료해 오세요. 당일 설치 때문에 실습 시간을 낭비하지 않으려면 이게 제일 중요합니다.

::: tip 시간이 얼마나 걸리나요?
순조롭게 진행되면 약 **20~30분** 안에 끝납니다. 혹시 막히는 부분이 있으면 아래 트러블슈팅을 먼저 보시고, 그래도 안 되면 당일 30분 일찍 오세요!
:::

- [ ] VS Code 설치
- [ ] Claude Code 설치 & 로그인 확인

## 📌 Step 1 of 4: Claude Pro 구독

::: warning 반드시 해야 합니다
Claude Code는 **Pro 이상 구독이 필수**입니다. 무료 계정으로는 실행되지 않습니다.
:::

Claude Code를 사용하려면 Claude **Pro 이상** 구독이 필요합니다. 무료 계정으로는 Claude Code를 실행할 수 없습니다.

1. [https://claude.ai](https://claude.ai) 접속
2. 계정이 없으면 **회원가입** (구글 계정으로 빠르게 가능)
3. 오른쪽 상단 프로필 → **Upgrade to Pro** → 결제
4. 월 $20 (약 27,000원). 언제든 해지 가능.

::: tip 이미 Pro이신 분
로그인 후 화면 왼쪽 하단에 "Claude Pro" 뱃지가 보이면 OK입니다. 바로 Step 2로 넘어가세요.
:::

::: info ✅ 여기까지 완료!
Claude Pro 구독이 되어 있다면 Step 1 끝입니다. 수고하셨어요!
:::

## 📌 Step 2 of 4: VS Code 설치

::: warning 반드시 해야 합니다
Claude Code를 가장 편하게 사용하는 편집기입니다. 코딩 몰라도 괜찮아요 — 오늘은 AI와 대화하는 창으로만 씁니다.
:::

### Mac

1. [https://code.visualstudio.com](https://code.visualstudio.com) 접속
2. **Download for Mac** 클릭
3. 다운로드된 `.zip` 파일 더블클릭 → `Visual Studio Code.app` 생성
4. `Visual Studio Code.app`을 **응용 프로그램(Applications) 폴더로 드래그**
5. 응용 프로그램에서 VS Code 실행

### Windows

1. [https://code.visualstudio.com](https://code.visualstudio.com) 접속
2. **Download for Windows** 클릭
3. 다운로드된 `.exe` 파일 실행
4. 설치 옵션은 **기본값 그대로** 다음 → 다음 → 설치
5. 설치 완료 후 VS Code 실행

**설치 확인:** VS Code를 열었을 때 아래와 같은 시작 화면이 보이면 성공입니다.

```
Welcome to Visual Studio Code
```

::: info ✅ 여기까지 완료!
VS Code 시작 화면이 보인다면 Step 2 끝입니다. 거의 다 왔어요!
:::

## 📌 Step 3 of 4: Claude Code 설치

::: tip 이건 안 해도 됩니다 (방법 B)
방법 A가 더 쉽습니다. 방법 A를 먼저 시도하고, 안 될 때만 방법 B로 넘어가세요.
:::

Claude Code를 설치하는 방법은 두 가지입니다. **방법 A가 더 쉽습니다.** 방법 A를 먼저 시도하세요.

### 방법 A: VS Code 확장으로 설치 (추천)

터미널을 열 필요 없이 VS Code 안에서 바로 설치하는 방법입니다.

1. VS Code 실행
2. 왼쪽 사이드바에서 **네모 4개 모양 아이콘** (Extensions) 클릭
   - 단축키: `Ctrl+Shift+X` (Windows) / `Cmd+Shift+X` (Mac)
3. 상단 검색창에 **`Claude Code`** 입력
4. 목록에서 **"Claude Code" — Anthropic** 찾기
5. **Install** 버튼 클릭
6. 설치 완료 후 왼쪽 사이드바에 **Claude 아이콘(사람 모양)** 이 생기면 성공

::: tip 아이콘이 안 보이면
설치 후 VS Code를 완전히 종료하고 다시 열어보세요. 그래도 없으면 왼쪽 사이드바 빈 공간에서 우클릭 → Claude Code 체크.
:::

---

### 방법 B: 터미널로 설치 (방법 A가 안 될 때)

::: warning 방법 A가 안 될 때만 이 방법을 사용하세요
이 방법은 Node.js가 필요합니다.
:::

**Node.js 먼저 설치 (없는 경우)**

1. [https://nodejs.org](https://nodejs.org) 접속
2. **LTS** 버전 다운로드 (왼쪽 버튼, 안정적인 버전)
3. 다운로드된 파일 실행 → 기본 옵션으로 설치

**Claude Code 설치 명령어**

- **Mac:** `터미널` 앱 열기 (Spotlight에서 "terminal" 검색)
- **Windows:** `PowerShell` 앱 열기 (시작 메뉴에서 "powershell" 검색)

터미널/PowerShell에 아래를 입력하고 Enter:

```bash
npm install -g @anthropic-ai/claude-code
```

설치가 완료되면 아래 명령어로 확인:

```bash
claude --version
```

버전 번호가 나오면 성공입니다.

::: info ✅ 여기까지 완료!
Claude 아이콘이 VS Code에 보이거나, 터미널에서 버전 번호가 나왔다면 Step 3 끝입니다!
:::

## 📌 Step 4 of 4: 로그인 확인

::: warning 반드시 해야 합니다
설치 후 반드시 로그인까지 확인해야 세미나 당일 바로 실습할 수 있습니다.
:::

설치가 끝났으면 실제로 작동하는지 확인합니다.

**VS Code 확장 방법으로 설치한 경우:**
1. VS Code 왼쪽 Claude 아이콘 클릭
2. **Sign in** 버튼 클릭 → 브라우저에서 Claude 계정으로 로그인
3. VS Code로 돌아와서 채팅창에 **`안녕!`** 입력
4. Claude가 응답하면 완료!

**터미널 방법으로 설치한 경우:**
1. 터미널에서 `claude` 입력 후 Enter
2. 처음 실행 시 로그인 프롬프트 → 브라우저 열림 → 로그인
3. 터미널로 돌아와서 **`안녕!`** 입력
4. 응답이 오면 완료!

::: tip 성공 기준
Claude가 "안녕하세요!" 같은 응답을 돌려주면 모든 준비가 끝난 겁니다.
:::

::: info ✅ 여기까지 완료!
Claude가 "안녕하세요!"라고 응답했다면 Step 4 끝입니다. 완벽해요!
:::

## 트러블슈팅 (자주 발생하는 문제)

막히는 게 생기면 아래를 먼저 확인해 보세요.

:::details `command not found: npm` 오류
**원인:** Node.js가 설치되어 있지 않습니다.

**해결:** [nodejs.org](https://nodejs.org)에서 LTS 버전을 설치한 뒤, 터미널을 완전히 종료하고 다시 열어 명령어를 입력하세요.
:::

:::details `permission denied` 오류
**원인:** 현재 계정에 설치 권한이 없습니다.

**해결:**
- Mac: 명령어 앞에 `sudo ` 를 붙여 입력 (예: `sudo npm install -g @anthropic-ai/claude-code`)
- Windows: PowerShell을 "관리자 권한으로 실행"하여 다시 시도
:::

:::details 로그인해도 "구독이 필요합니다" 메시지
**원인:** Claude Pro 구독이 되어 있지 않습니다.

**해결:** [claude.ai](https://claude.ai) 접속 후 왼쪽 하단의 구독 상태를 확인하고, Pro로 업그레이드하세요.
:::

:::details VS Code 확장 검색에 Claude Code가 안 나옴
**원인:** 검색어에 오타가 있거나 인터넷 연결이 불안정합니다.

**해결:** `Claude Code` 를 정확히 입력하세요 (대소문자 무관, 공백 포함). 그래도 안 나오면 VS Code를 재시작 후 다시 시도하세요.
:::

:::details 브라우저 로그인 후 VS Code에 반응 없음
**원인:** 브라우저 팝업이 차단되었습니다.

**해결:** 브라우저 주소창 오른쪽의 팝업 차단 아이콘을 클릭해 허용으로 바꾼 뒤 다시 시도하세요.
:::

:::details Mac에서 "개발자를 확인할 수 없음" 경고
**원인:** macOS 보안 설정이 앱 실행을 막고 있습니다.

**해결:** 시스템 설정 → 개인 정보 보호 및 보안 → 화면 하단 "그래도 열기" 클릭
:::

## 안 되면?

막히는 부분이 있어도 걱정하지 마세요. 두 가지 방법으로 해결할 수 있습니다.

**방법 1: 당일 일찍 오기**
세미나 시작 30분 전에 오시면 1:1로 직접 도와드립니다. 이 방법이 제일 빠릅니다.

**방법 2: 사전에 스크린샷 보내기**
오류 화면을 캡처해서 카톡으로 보내주시면 미리 해결책을 알려드립니다.

::: warning 당일 설치는 피해주세요
당일 설치가 시작되면 다른 참가자분들의 실습 시간이 줄어듭니다. 꼭 전날까지 완료해 주세요. 정말 안 되시면 일찍 오시는 방법을 이용해 주세요!
:::
