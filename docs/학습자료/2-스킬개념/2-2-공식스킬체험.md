직접 만들기 전에, **남이 만든 스킬**을 먼저 써봅니다.

- 스킬을 설치하는 두 가지 방법을 익힌다
- **slack-gif-creator로 GIF를 직접 만들어본다**
- "스킬 하나로 이런 것까지 되는구나!" 를 체감한다

## 공식 스킬이란?

Claude를 만든 **Anthropic**이 직접 만들어서 GitHub에 공개한 스킬입니다.
누구나 무료로 다운받아 쓸 수 있어요.

::: info 공식 스킬 저장소
Anthropic의 GitHub에서 다양한 공식 스킬을 확인할 수 있습니다.
문서 처리, 디자인, 업무 자동화 등 카테고리별로 정리되어 있어요.
:::

### 추천 스킬 카테고리

| 카테고리 | 대표 스킬 | 하는 일 | 난이도 |
|---------|----------|--------|:------:|
| 문서 처리 | `pdf` | PDF 읽기, 요약, 분석 | 쉬움 |
| 디자인 | `slack-gif-creator` | Slack용 애니메이션 GIF 만들기 | 쉬움 |
| 업무 | `meeting-summary` | 회의록 자동 요약 | 쉬움 |
| 데이터 | `xlsx` | 엑셀 파일 읽기/분석 | 보통 |
| 프레젠테이션 | `pptx` | PPT 파일 생성/수정 | 보통 |

## 실습 1: 수동 설치 (폴더 이해용)

스킬이 실제로 어떤 구조인지 이해하기 위해, 먼저 **수동으로** 설치해봅니다.

### Step 1: GitHub에서 스킬 다운로드

1. Anthropic 스킬 저장소 방문
2. 원하는 스킬 폴더 찾기 (예: `slack-gif-creator/`)
3. 폴더 안의 `SKILL.md` 파일 확인

### Step 2: 내 스킬 폴더에 복사

```
다운받은 스킬 폴더
      ↓ 복사
~/.claude/skills/slack-gif-creator/SKILL.md
```

::: details Finder로 복사하는 방법 (Mac)
1. Finder 열기
2. `Cmd + Shift + G` 누르기
3. `~/.claude/skills/` 입력 후 이동
4. skills 폴더 안에 다운받은 스킬 폴더를 드래그
5. 완료!

skills 폴더가 없다면? Claude Code에게 "~/.claude/skills/ 폴더 만들어줘"라고 시키세요.
:::

### Step 3: 설치 확인

Claude Code 채팅창에 입력:
```
사용 가능한 스킬 목록을 보여줘
```

방금 설치한 스킬이 목록에 보이면 성공!

---

## 더 쉬운 방법: 명령어 한 줄

수동 설치는 구조를 이해하기 위한 것이었고, 실제로는 **명령어 한 줄**이면 됩니다.

Claude Code 채팅창에:
```
slack-gif-creator 스킬을 설치해줘
```

또는 공식 스킬 마켓에서:
```
/install-skill slack-gif-creator
```

::: tip 어떤 방법이든 결과는 같습니다
수동이든 명령어든, 결국 `~/.claude/skills/스킬이름/SKILL.md` 파일이 생기는 것은 동일합니다.
편한 방법을 쓰세요!
:::

## 실습 2: slack-gif-creator로 GIF 만들기

이제 오늘의 하이라이트입니다! **스킬 하나로 애니메이션 GIF를 만들어봅시다.**

### 왜 GIF 스킬인가요?

- 눈에 보이는 결과물 → "와, 스킬로 이런 것까지?" 체감
- Slack, 카톡, 이메일에 바로 활용 가능
- 만드는 과정이 재밌음!

### Step 1: 스킬 설치 확인

아까 설치했다면 바로 사용 가능합니다. 확인해볼까요?

```
/slack-gif-creator 안녕하세요 텍스트가 반짝이는 GIF 만들어줘
```

### Step 2: 다양한 GIF 만들어보기

::: info 지금 해보세요! 아래 중 마음에 드는 걸 골라서 입력해보세요.
:::

**예시 1: 축하 GIF**
```
/slack-gif-creator 🎉 축하합니다! 텍스트가 위에서 떨어지는 애니메이션 GIF
```

**예시 2: 업무용 GIF**
```
/slack-gif-creator "수고하셨습니다" 라는 텍스트에 별이 반짝이는 GIF
```

**예시 3: 자유 주제**
```
/slack-gif-creator [여러분이 원하는 내용을 자유롭게!]
```

### Step 3: 결과물 확인

GIF 파일이 생성되었나요? 파일을 열어서 확인해보세요.

::: tip 이걸 내 업무용으로도 만들 수 있다!
지금은 재미로 GIF를 만들었지만, 같은 원리로:
- 매주 보내는 주간보고를 자동화하거나
- 고객 응대 이메일을 템플릿화하거나
- 회의록을 자동으로 정리하는 스킬도 만들 수 있습니다

**스킬의 핵심은 "SKILL.md 파일 하나"라는 것, 기억하시죠?**
:::

## 스킬 작동 원리 다이어그램

지금까지 해본 걸 정리하면 이런 흐름입니다:

```
① 사용자: "/slack-gif-creator 축하 GIF 만들어줘"
        ↓
② Claude Code: description 확인 → "GIF 만들기 스킬이구나"
        ↓
③ SKILL.md 전문 읽기 → 출력 형식, 규칙, 도구 파악
        ↓
④ 지침대로 GIF 생성 코드 작성 & 실행
        ↓
⑤ 결과: animated.gif 파일 생성!
```

::: info 핵심 포인트
모든 스킬은 이 흐름을 따릅니다. GIF든, 이메일이든, 보고서든 — 패턴은 동일합니다.

1. **사용자가 요청** → 2. **description으로 스킬 매칭** → 3. **SKILL.md 읽기** → 4. **실행** → 5. **결과물**
:::

## 문제 해결

| 문제 | 원인 | 해결 |
|------|------|------|
| "스킬을 찾을 수 없습니다" | 파일 위치가 잘못됨 | `~/.claude/skills/slack-gif-creator/SKILL.md` 경로 확인 |
| 스킬이 실행은 되는데 GIF가 안 만들어짐 | 의존성 부족 | Claude Code에게 "필요한 패키지 설치해줘"라고 요청 |
| GIF가 만들어졌는데 파일을 못 찾겠음 | 저장 경로 확인 | Claude Code에게 "방금 만든 GIF 파일 위치 알려줘"라고 요청 |
| 아무 반응이 없음 | Claude Code 재시작 필요 | Claude Code를 껐다 다시 켜기 |
