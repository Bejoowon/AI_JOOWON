# 클로드코드란?

처음 들어보셨나요? 괜찮습니다. AI가 뭔지부터 차근차근 설명해 드릴게요.

---

## AI가 뭔가요?

**인공지능(AI)** 은 사람처럼 생각하고 대화할 수 있는 컴퓨터 프로그램입니다. 요즘 많이 들리는 ChatGPT, Gemini 같은 게 다 AI예요. 질문하면 답해주고, 글을 써주고, 요약도 해줍니다.

---

## Claude란?

**Claude(클로드)** 는 **Anthropic(앤트로픽)** 이라는 미국 회사가 만든 AI입니다.

ChatGPT를 만든 OpenAI의 경쟁자라고 생각하시면 됩니다. 둘 다 "대화형 AI"인데, Claude만의 특징이 있습니다:

| 특징 | 설명 |
|------|------|
| **긴 글 이해** | 책 한 권 분량의 텍스트도 한 번에 읽고 분석 |
| **안전성 중시** | "해로운 답변을 안 하는 것"에 특히 신경 씀 |
| **자연스러운 한국어** | 한국어 대화와 문서 작성 품질이 뛰어남 |
| **코드 작성 능력** | 프로그래밍 실력이 업계 최상위급 |

::: info Claude를 써보고 싶다면
[claude.ai](https://claude.ai)에서 무료로 대화해볼 수 있습니다. ChatGPT처럼 웹 브라우저에서 채팅하는 방식이에요.
:::

---

## Claude Code란?

> **Claude Code = Claude를 내 컴퓨터에서 직접 쓰는 도구**

claude.ai에서 채팅하는 건 "전화 상담"이에요. 내 파일을 직접 보여줄 수 없고, 결과물도 복사-붙여넣기 해야 하죠.

Claude Code는 다릅니다. AI가 **내 컴퓨터 안에 들어와서** 직접 파일을 열고, 만들고, 수정합니다.

```
채팅 AI (claude.ai, ChatGPT)
  → "이거 해줘" → 결과를 복붙 → 내가 직접 적용

Claude Code
  → "이거 해줘" → AI가 직접 파일을 만들고 수정 → 끝!
```

::: tip 한마디로
**"복붙 없이, 매번 새로 설명 없이, AI가 내 컴퓨터에서 알아서 해주는 경험"** 이 Claude Code입니다.
:::

---

## Claude vs Claude Code 차이

|  | Claude (웹) | Claude Code |
|---|------------|-------------|
| **어디서?** | 웹 브라우저 (claude.ai) | 내 컴퓨터 (VS Code / 터미널) |
| **뭘 하나?** | 대화, 질문 답변, 글쓰기 | 내 파일을 직접 읽고, 만들고, 수정 |
| **비유하면** | 전화 상담사 | 옆자리 동료 |
| **파일 접근** | 대화창에 복붙해야 함 | 내 폴더를 직접 봄 |
| **자동화** | 매번 새로 요청해야 함 | **스킬**로 반복 실행 가능 |
| **외부 연결** | 제한적 | **MCP**로 캘린더, DB 등 연결 |
| **기억** | 대화 끝나면 리셋 | **CLAUDE.md**로 영구 기억 |

![ChatGPT는 전화 상담사, Claude Code는 옆자리 동료](/images/chatgpt-vs-claude-code.png)

::: info 좀 더 생생하게 비유해 볼게요

**Claude (웹) = 전화 상담사**
전화로 "제 냉장고 안에 뭐가 있는지 알려줘"라고 물어봐도, 상담사는 여러분 집 냉장고를 볼 수가 없죠. 여러분이 직접 냉장고 문을 열어서 "달걀 3개, 우유 1개…" 읽어줘야 합니다.

**Claude Code = 옆자리 동료**
"냉장고 좀 봐줄 수 있어?"라고 하면, 동료가 직접 냉장고 문을 열고 확인해 줍니다. 필요하면 장볼 목록도 바로 작성해 주고, 내일 일정과 비교해서 "이번 주 저녁은 이걸로 하면 어때요?"까지 제안해 줍니다.
:::

## 비슷한 서비스 비교

"이런 AI 코딩 도구가 Claude Code만 있는 건 아니잖아요?" 맞습니다. 간단히 비교해볼게요.

| 서비스 | 만든 곳 | 특징 | 가격 |
|--------|---------|------|------|
| **Claude Code** | Anthropic | 스킬(자동화), CLAUDE.md(기억), MCP(외부연결) | Pro $20/월 |
| **ChatGPT** | OpenAI | 가장 유명한 AI. 웹 채팅 중심 | Plus $20/월 |
| **GitHub Copilot** | Microsoft | 코드 자동완성 특화. 개발자 도구 | $10/월~ |
| **Cursor** | Cursor Inc. | VS Code 기반 AI 코드 편집기 | $20/월~ |
| **Windsurf** | Codeium | AI 코딩 에디터. Cursor 대안 | $15/월~ |

::: tip 비개발자에게는?
GitHub Copilot, Cursor, Windsurf는 **개발자 특화** 도구입니다.
비개발자가 업무 자동화 목적으로 쓰기에는 **Claude Code가 가장 접근하기 쉽습니다.**
스킬을 글로 적으면 되니까요 — 코드를 몰라도 됩니다.
:::

## 왜 Claude Code인가?

Claude Code만의 3가지 무기가 있습니다. 이 가이드에서 하나씩 배워봅니다.

### 1. 스킬 (Skills) — 반복 업무 자동화

> "한 번 만들면 `/스킬이름` 한 줄로 매번 재사용"

매주 같은 형식의 보고서, 매번 비슷한 이메일... 스킬로 만들면 1분이면 끝납니다.

### 2. CLAUDE.md — AI의 기억

> "한 번 알려주면 매번 기억"

"한국어로, 존댓말로, 보고서 형식은 이거로..." 매번 설명할 필요 없습니다.

### 3. MCP — 외부 도구 연결

> "캘린더, 노션, 슬랙까지 AI가 직접 조작"

"내일 일정 잡아줘"가 실제로 Google Calendar에 반영됩니다.

::: tip 성장 순서
**스킬 → CLAUDE.md → MCP**

오늘은 스킬에 집중합니다. 나머지는 스킬을 마스터한 뒤 자연스럽게 배우게 됩니다.
:::

## 무엇을 할 수 있나요?

실제 상황으로 보여드릴게요.

### Before / After

**상황 1: 거래처 이메일 작성**

| Before (기존 방식) | After (Claude Code) |
|----------------------|--------------------------|
| ChatGPT 열기 → 상황 설명 → 초안 복사 → 이메일 앱에 붙여넣기 | "거래처에 납품 일정 변경 이메일 써줘" → 파일로 바로 저장 |

**상황 2: 회의록 정리**

| Before (기존 방식) | After (Claude Code) |
|----------------------|--------------------------|
| 회의록 복사 → ChatGPT에 붙여넣기 → 요약 복사 → 문서에 붙여넣기 | "회의록 요약해줘" → 요약본 파일 자동 생성 |

**상황 3: 반복 업무 자동화**

| Before (기존 방식) | After (Claude Code) |
|----------------------|--------------------------|
| 매주 월요일마다 같은 내용을 처음부터 다시 입력 | 스킬 한 번 만들어두면 `/주간보고` 한 줄로 끝 |

::: details 개발자가 아닌데 써도 될까요?
물론입니다! Claude Code는 원래 개발자용으로 만들어졌지만, 문서 작업, 데이터 정리, 이메일 작성, 일정 관리 같은 일반 업무에도 매우 유용합니다. 이 가이드가 바로 그 사례를 보여드리기 위해 준비됐습니다.
:::

## 어떻게 생겼나요?

Claude Code는 두 가지 방식으로 사용합니다.

**VS Code 안에서 사용할 때**

VS Code(코드 편집기) 왼쪽 사이드바에 Claude 아이콘이 생깁니다. 클릭하면 채팅창이 열리고, 거기서 AI에게 지시를 내리면 됩니다. AI는 현재 열려 있는 폴더의 파일들을 직접 보고 수정합니다.

**터미널에서 사용할 때**

터미널(검은 화면의 명령어 입력창)에서 `claude` 라고 입력하면 Claude Code가 실행됩니다. 마찬가지로 채팅 형식으로 대화하면 됩니다.

::: tip 터미널이 낯설어도 괜찮아요
이 가이드에서는 VS Code 사이드바 방식을 주로 사용합니다. 터미널을 몰라도 충분히 따라올 수 있습니다.
:::

## 핵심 정리

::: tip 기억하세요
- **AI** = 사람처럼 대화하는 컴퓨터 프로그램
- **Claude** = Anthropic이 만든 AI (ChatGPT의 경쟁자)
- **Claude Code** = Claude를 내 컴퓨터에서 직접 쓰는 도구

**대화형 AI** + **내 컴퓨터 접근** + **자동화** = Claude Code

ChatGPT가 "말로 도와주는 AI"라면, Claude Code는 "손발이 있는 AI"입니다.
:::
