- "이 부분이 아쉽다" 싶은 게 생기면 → 그게 다음 개선 포인트입니다

**이걸 하면 이런 변화가!**

::: info 잘 안 되는 게 정상입니다

- 테스트 → 결과 확인 → 수정 → 반복

**이걸 하면 이런 변화가!**
> 나만의 두 번째 스킬이 생깁니다.
> "내가 AI를 커스터마이징 했다" 는 뿌듯함을 느끼게 됩니다.

### 📅 2주 후 — 3단계: CLAUDE.md 작성하기

**이걸 하면 이런 변화가!**
